# Sketch Nomad

**Sketch Nomad** is a simple platform where outdoor artists can share their artwork with location tags and explore art from others.

## What is Sketch Nomad?

A community for outdoor artists to share location-tagged artwork and discover what others are creating. Browse art organized by artist and city, connecting you with the creative work happening around the world.
This repository contains the promotional website ([sketchnomad.com](https://sketchnomad.com)). The main web application is available at [sketchnomad.app](https://sketchnomad.app).


## Development

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

Visit `http://localhost:4321` to view the site locally.
